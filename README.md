User stories
-------------
As a user
So that I can view my favourite bookmarks
I would like to see a list of bookmarks.

As a user
So that I can save a favourite website
I would like to add a favourite website to bookmarks.

As a user
So I can delete a bookmark I no longer need
I want to delete a bookmark in Bookmark Manager

As a user
So I can change a bookmark in Bookmark Manager
I want to update a bookmark
